# Week 0: Querying

These exercises focus on **retrieving, filtering, and summarizing data** from various databases.

## Key Topics

- Understanding Database Management Systems (DBMS)
- Writing basic `SELECT` queries to retrieve specific columns
- Filtering data with `WHERE` conditions, including `AND`, `OR`, `NOT`, and `NULL`
- Pattern matching with `LIKE`
- Sorting results using `ORDER BY` and limiting output with `LIMIT` and `OFFSET`
- Using aggregate functions (`COUNT`, `ROUND`, `DISTINCT`, etc.) to summarize data

## Purpose of the Exercises

- **Cyberchase**: Querying a real-world dataset of TV episodes, practicing filtering, ordering, and aggregation
- **Normals**: Exploring scientific datasets, working with numeric data, ranges, and averages
- **Views**: Analyzing artwork data using filtering, ordering, and aggregate functions (brightness, contrast, entropy, etc.)
- **Players**: Applying querying skills to a historical sports dataset, extracting insights about playersâ€™ statistics and attributes

> The `.sql` files each contain a single query that solves a specific problem.
